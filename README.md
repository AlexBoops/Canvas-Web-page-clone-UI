# 🎓 Canvas Clone UI  

A **front-end project** replicating the user interface of the popular **Canvas Learning Management System (LMS)**.  
This project recreates the dashboard experience with a clean and responsive design.

## ✨ Features  
- 🔑 **Login Page** – Styled authentication page.  
- 📚 **Dashboard** – Displays enrolled courses in card layout.  
- 📂 **Course Cards** – Each course includes thumbnail, title, and quick links.  
- 📌 **Sidebar Navigation** – Links for Dashboard, Courses, Calendar, Inbox, etc.  
- 👤 **Header with Profile Menu** – Includes notifications and user account menu.  
- 📱 **Responsive Design** – Optimized for desktop, tablet, and mobile.  

---

## 🛠️ Tech Stack  
- **HTML5** – Page structure  
- **CSS3 / TailwindCSS / Bootstrap** – Styling and responsiveness  
- **JavaScript** – Dynamic interactivity
